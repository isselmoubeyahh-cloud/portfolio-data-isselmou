# Portfolio Data – Isselmou Beyah

Ce dépôt regroupe des travaux en statistique, économétrie et data science (Python/SQL/Power BI) réalisés dans un cadre académique et appliqué.

## Compétences mises en avant
- Analyse statistique et économétrique (descriptif, tests, régression)
- Nettoyage et préparation de données
- Data visualisation et restitution (rapports, dashboards)
- Machine learning 

## Outils
Python (pandas, numpy, statsmodels, scikit-learn, matplotlib), SQL, Power BI.

## Confidentialité des données
Certaines données ne peuvent pas être publiées. Dans ce cas, le dépôt contient le code, des figures et des synthèses, ainsi qu’un fichier expliquant comment reproduire l’analyse avec des données similaires.
